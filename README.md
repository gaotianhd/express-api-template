# express-api-template

#### 介绍
express-api 模板项目

#### 软件架构
软件架构说明  
|--api  
|  |--index  api入口文件  
|  |--modules  api模块文件夹  
|    |--aes  aes加密解密  
|    |--aliOss  阿里对象存储示例  
|    |--sha256  sha256加密  
|    |--testApi  测试api文件  
|--models  
|  |--index  models入口文件  
|  |--modules  models模块文件夹  
|    |--testModels  测试models文件  
|--routes  
|  |--index  路由入口文件  
|  |--modules  路由模块文件夹  
|    |--testRoutes  测试路由文件  
|--config  配置文件  
|--middleware  中间件文件  
|--redis  redis连接文件  
|--server  启动服务文件  
|--utils  工具文件  
|--entry  项目入口文件
